# Mini-Git in C

Проект по программированию на C.

## Тема

Мини-Git: упрощённая персистентная система контроля версий текстовых файлов.

## Идея

Программа хранит историю неизменяемых коммитов. Каждый коммит содержит список файлов и ссылки на blob-объекты с содержимым. Если файл не изменился между коммитами, его содержимое не копируется заново, а переиспользуется по хешу.

## Условия

Условия выполнения описаны в файле miniGit.pdf

## Сборка

Для спокойствия make clean, после чего make

## Описание команд

| Команда | Описание |
|---|---|
| `./minigit init` | Создает каталог |
| `./minigit add <file>` | Добавить или обновить файл |
| `./minigit rm <file>` | Отметить файл для удаления |
| `./minigit status` | Показать текущий статус изменений |
| `./minigit commit "<message>"` | Создать коммит |
| `./minigit log` | Показать историю коммитов |
| `./minigit files` | Показать файлы в текущем коммите |
| `./minigit files <commit_id>` | Показать файлы в выбранном коммите |
| `./minigit show <commit_id> <file>` | Вывести файл из коммита |
| `./minigit exists <commit_id> <file>` | Проверка файлы на существование в выбранном коммите |
| `./minigit commit-info <commit_id>` | Информация о коммите |
| `./minigit stats` | Вывести статистику |
| `./tests/manual_test.sh` | Провести тестирование всех функций |
