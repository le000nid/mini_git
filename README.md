# Mini-Git in C

Проект по программированию на C.

## Тема

Мини-Git: упрощённая персистентная система контроля версий текстовых файлов.

## Идея

Программа хранит историю неизменяемых коммитов. Каждый коммит содержит список файлов и ссылки на blob-объекты с содержимым. Если файл не изменился между коммитами, его содержимое не копируется заново, а переиспользуется по хешу.

## Планируемые команды

```bash
./minigit init
./minigit add <file>
./minigit rm <file>
./minigit commit "<message>"
./minigit log
./minigit files
./minigit show <commit_id> <file>
./minigit stats
